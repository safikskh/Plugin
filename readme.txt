# custom slider

This slider is custom slider and using a js and css it's a wordpress backend editable.

## Installation

Use the package manager  to install custom slider.



## Usage


Using [show_slider] short code you can display slider where you want to add.

##  jQuery slideshow library

### Slick slider(for slider)
I have used the slick slider library (https://kenwheeler.github.io/slick/)

### Change order of images
I have used the jQuery sort-able library (http://jqueryui.com/demos/sortable/#connect-lists)
